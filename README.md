# Teste técnico

Monorepo com **`/parte-1`** (Figma → código) e **`/parte-2`** (hero landing curso Python/IA). **Stack:** Next.js 16, React 19, TypeScript, Tailwind CSS 4, Motion; na parte 2 também `@phosphor-icons/react`.

- **Deploy parte 1 (site):** https://teste-tau-black.vercel.app/
- **Deploy parte 2 (hero):** https://teste-4iww.vercel.app/

**Ferramentas de IA:** Cursor (assistente) e **MCP do Figma** para extrair a base do layout direto do arquivo. **Etapas em que a IA ajudou mais:** montar o **layout inicial** (estrutura e blocos) com mais rapidez. **Ajustes manuais:** polir **espaçamentos, tipografia, fontes e imagens** até ficar alinhado ao design; na parte 2, composição da hero e refinamento visual sem Figma fixo.
